# DevManager

Integrate gitlab issues board + clockify + pomodoro (local) manage time can sync with clockify  
  
Description files:  
 - `index.html` is main file have html css js, this can deploy to PWA or load as chrome extension,   
 - `manifest.json` for chrome extension,  
 - `background.js` chrome extension running background,  
 - `manifest.webmanifest` manifest PWA,  
 - `sw.js` service worker PWA
